## **Installing Apache Server on Ubuntu**

Apache HTTP Server (httpd) is a widely used web server for hosting web applications.

---

## **Install Apache**

Update package list and install Apache:

```bash
sudo apt update
sudo apt install apache2 -y
```

---

## **Start / Stop / Restart Apache**

Use the following commands to manage the Apache service:

```bash
sudo systemctl start apache2
sudo systemctl stop apache2
sudo systemctl restart apache2
sudo systemctl reload apache2
```

* **start** → Starts Apache service
* **stop** → Stops Apache service
* **restart** → Restarts Apache service
* **reload** → Reloads configuration without downtime
